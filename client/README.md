# 🚀 FinPilot — Smart Personal Finance Dashboard

> A modern, intuitive, and visually rich personal finance dashboard to track income, expenses, and savings with ease.

---

## ✨ Overview

**FinPilot** is a responsive financial dashboard built to help users manage their transactions efficiently.
It combines **clean UI, smooth animations, and smart data visualization** to deliver a premium product-like experience.

---

## 🎯 Features

* 📊 **Transaction Management**

  * Track income, expenses, and savings
  * Categorized financial records
  * Sorted by latest activity

* 🎨 **Modern UI/UX**

  * Smooth animations using Framer Motion
  * Fully responsive layout

* ⚡ **High Performance**

  * Optimized rendering using React hooks (`useMemo`)
  * Efficient state management with Zustand


* 🔍 **Smart Filtering**

  * Filter transactions by type (Income / Expense / Savings)

* 📈 **Scalable Architecture**

  * Easily extendable for backend integration
  * Mock data generation for realistic testing
